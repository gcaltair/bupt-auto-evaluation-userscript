# 北邮自动评教油猴脚本

这是一个面向北京邮电大学教务系统的 Tampermonkey / 油猴脚本。脚本会在评教页面生成悬浮按钮，点击后自动选择正向选项、随机勾选若干正面评语，并填入随机主观评语。

脚本不会自动保存或提交评教结果。请务必在点击系统里的“保存”或“提交”前自行检查内容。

## 功能

- 自动选择每个评项所在行的第一个单选项，通常对应“完全符合”
- 随机勾选 3 到 5 个正面评语选项
- 随机填充“建议”和“亮点”等文本框
- 兼容普通教务路径和 `webvpn.bupt.edu.cn` VPN 页面
- 支持 iframe 页面，适配强智系统常见页面结构

## 安装

1. 安装浏览器扩展 [Tampermonkey](https://www.tampermonkey.net/)。
2. 打开脚本 raw 地址：
   <https://raw.githubusercontent.com/gcaltair/bupt-auto-evaluation-userscript/main/bupt-auto-evaluation.user.js>
3. 在 Tampermonkey 安装页面点击安装。

## 使用

1. 登录北京邮电大学教务系统或 WebVPN 中的教务系统。
2. 进入评教页面。
3. 点击右上角的“随机一键好评”按钮。
4. 检查自动填充的评分、勾选项和文字评语。
5. 确认无误后，再手动点击系统页面的保存或提交按钮。

## 适配范围

脚本匹配以下页面：

```text
*://*/*jsxsd/*
*://webvpn.bupt.edu.cn/*
```

## 隐私与安全

- 脚本不请求网络权限。
- 脚本不收集、上传或保存任何个人信息。
- 脚本只在当前页面内操作表单元素。
- 脚本不会自动提交评教表单。
